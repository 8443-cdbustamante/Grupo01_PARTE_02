Animaciones DOM

En HTML se pueden incorporar animaciones usando en conjunto CSS y Javascript. Hay que recordar utilizar un elemento DIV como contenedor, fijando la posición en relativa.La animación irá en un elemento DIV de animación, fijado en posición absoluta. En Javascript se coloca la programación de la animación, que en su norma general utilizan temporizadores para los cambios graduales de los elementos a animar. 


    <!doctype html>
    <html lang="en-US">
    
    <head>
    <meta charset="utf-8">
    <title>Web Animations API implicit keyframes</title>
    <style>
    html {
    height: 100%;
    }
    
    body {
    margin: 0;
    height: inherit;
    display: flex;
    justify-content: center;
    align-items: center;
    }
    
    div {
    width: 150px;
    height: 100px;
    background-color: red;
    border: 10px solid black;
    border-radius: 10px;
    }
    </style>
    </head>
    
    <body>
    <div>
    
    </div>
    
    <script>
    const divElem = document.querySelector('div');
    
    let rotate360 = [
    { transform: 'rotate(360deg)' }
    ];
    
    let slowInfinite = {
    duration: 5000,
    iterations: Infinity
    }
    
    divElem.animate( rotate360, slowInfinite );
    
    </script>
    </body>
    
    </html>

Event Listener

Se refiera a la utilización del metodo "addEventListener()" para controlar eventos de un elemento específico de forma ordenada y evitando sobreescribir eventos. Estos trabajan con cualquier elemento DOM de la ventana. En el ejemplo siguiente podemos ver como iteractuan 2 event listeners. 

  <!doctype html>
    <html lang="en-US">
    
    <head>
    <meta charset="utf-8">
    <title>Web Animations API implicit keyframes</title>



    </head>
    
    <body>

<button id="example-button">You have not clicked this button.</button>
<button id="reset-button">Click this button to reset the first button.</button>



    
    <script>
    const buttonToBeClicked = document.getElementById("example-button");

const resetButton = document.getElementById("reset-button");

// the text that the button is initialized with
const initialText = buttonToBeClicked.textContent;

// the text that the button contains after being clicked
const clickedText = "You have clicked this button.";

// we hoist the event listener callback function
// to prevent having duplicate listeners attached
function eventListener() {
  buttonToBeClicked.textContent = clickedText;
}

function addListener() {
  buttonToBeClicked.addEventListener(
    "click",
    eventListener,
    {
      passive: true,
      once: true
    }
  );
}

// when the reset button is clicked, the example button is reset,
// and allowed to have its state updated again
resetButton.addEventListener(
  "click",
  () => {
    buttonToBeClicked.textContent = initialText;
    addListener();
  }
);

addListener();
    </script>
    </body>
    
    </html>

  </div>

  <script>
const parent = document.body.querySelector(".parent");

const addChild = document.body.querySelector("#add-child");
addChild.addEventListener("click", () => {
  // Only add a child if we don't already have one
  // in addition to the text node "parent"
  if (parent.childNodes.length > 1) {
    return;
  }
  const child = document.createElement("div");
  child.classList.add("child");
  child.textContent = "child";
  parent.appendChild(child);
});

const removeChild = document.body.querySelector("#remove-child");
removeChild.addEventListener("click", () => {
  const child = document.body.querySelector(".child");
  parent.removeChild(child);
});

  </script>


</body>

Colecciones HTML DOM

Las colecciones HTML DOM son grupos de elementos HTML que se pueden seleccionar y manipular de diferentes maneras en una página web. Por ejemplo, puedes usar una colección para seleccionar todos los elementos de una determinada etiqueta (como todos los elementos ` < p>` en una página) o todos los elementos con un determinado atributo (como todos los elementos con un atributo class="foo").

Las colecciones HTML DOM son una parte importante de la programación web, ya que te permiten trabajar con conjuntos de elementos en lugar de tener que seleccionar y manipular cada elemento individualmente. Esto te permite hacer cosas como cambiar el estilo de todos los elementos ` < p>` de una página con una sola línea de código, en lugar de tener que escribir código individualmente para cada elemento ` < p>`.

Hay varias colecciones HTML DOM disponibles, dependiendo del tipo de elementos que desees seleccionar y manipular. Algunas de las más comunes incluyen:

document.images: una colección de todas las imágenes en una página web.
document.links: una colección de todos los enlaces en una página web.
document.forms: una colección de todos los formularios en una página web.
document.getElementsByTagName(): una colección de todos los elementos HTML con una determinada etiqueta (como todos los elementos ` < p>`).
document.getElementsByClassName(): una colección de todos los elementos HTML con una determinada clase (como todos los elementos con un atributo class="foo").

Para usar una colección HTML DOM, primero debes seleccionarla y luego puedes trabajar con sus elementos de diferentes maneras. Por ejemplo, puedes recorrer una colección con un bucle `for` y luego modificar cada elemento de la colección individualmente. También puedes usar métodos como `add()` y `remove()` para agregar o eliminar elementos de la colección.

En resumen, las colecciones HTML DOM son una herramienta muy útil para seleccionar y manipular grupos de elementos en una página web de manera eficiente y sencilla.


Listas HTML DOM

Las listas en HTML se crean con la etiqueta ` < ul>` (para una lista no ordenada) o ` < ol>` (para una lista ordenada). Dentro de estas etiquetas se pueden añadir elementos de la lista con la etiqueta ` < li>`.

Por ejemplo, para crear una lista de compras podrías usar el siguiente código:


    < li>Leche
    < li>Huevos
    < li>Pan
    < li>Agua


Esto crearía una lista que se vería así: Leche Huevos Pan Agua Las listas en HTML pertenecen al DOM (Document Object Model), que es la estructura en la que se representa un documento HTML. El DOM permite manipular los elementos de una página web y es utilizado por los lenguajes de programación de la web, como JavaScript.


Navegación HTML DOM



El DOM (Modelo de Objeto de Documento) es una interface de programación que permite a los desarrolladores interactuar con un documento HTML o XML. Con el DOM, puedes modificar el contenido y el estilo de una página web dinámicamente, sin tener que recargar la página.

La navegación del DOM se refiere a la forma en que se pueden acceder y manipular los elementos de un documento HTML a través de la interfaz del DOM. Por ejemplo, puedes usar el DOM para buscar un elemento en particular en una página web y luego cambiar su contenido o estilo.

Para realizar una navegación del DOM en HTML, puedes usar varios métodos y propiedades del DOM, como getElementById() y querySelector().

Por ejemplo, supongamos que queremos cambiar el contenido de un elemento de tipo "p" en una página HTML.

Primero, podemos utilizar un método como "getElementById" para acceder al elemento que queremos modificar, y luego utilizar la propiedad "innerHTML" para cambiar su contenido.

La navegación de tipo DOM es una herramienta importante para los desarrolladores web, ya que les permite crear páginas dinámicas y adaptables a las necesidades de sus usuarios.
