JS HTML DOM

Conociendo los principios básicos

Introducción

Las cualidades de HTML DOM permite acceder y cambiar todos los elementos de un documento HTML, utilizando JavaScript.


Características

  • Creación, eliminación y cambios de elementos HTML
  • Creación, eliminación y cambios de atributos HTML
  • Gestión de estilos CSS
  • Creacióny gestión de eventos HTML

DOM

Denominado como Modelo de Objetos de Documento, es una plataforma para la actualización dinámica de contenido, estructura y estilo de documentos como XML y HTML.

HTML DOM

Se trata de un modelo estándar que trabaja con objetos, propiedades, métodos y eventos dentro de una página HTML

Métodos

En HTML DOM los métodos son las acciones que puede realizar un elemento, el cuál es considerado como un objeto al manejar en Javascript. Además dicho objetos tiene propiedades, valores a modificar propio del objeto.

Método getElementByID

Nos permite acceder a un elemento HTML usando su id definido

Propiedad innerHTML

Permite obtener o reemplazar el contenido de un elemento HTML

Componentes

Dentro de los documentos HTML, para acceder a un elemento hay que recordar acceder primero al objeto para trabajar en el contenido de los elementos

Cambiar elementos HTML.

Elementos

Para la identificación de los diferentes elementos que se encuentran dentro de un documento HTML se utiliza los id, nombre de la etiqueta, nombre de la clase, selectores CSS y colecciones de objetos HTML.


Ejemplo modificado

HTML

Se debe tomar en cuenta las principales propiedades para manipular el contenido HTML. innerHTML nos permite cambiar el contenido a uno igual. Además con ."atributo" podemos asignar un nuevo valor al atributo que desamos cambiar. Lo principal es que se puede incoroporar funcione JavaScript dentro de los elementos.


CSS

En las hojas de estilos podemos cambiar sus valores al señalar el objeto, seguido de .style."propiedad", eligiendo la propiedad requerida para cambiar su valor. Dentro de CSS también se puede manejar eventos, como por ejemplo cuando se hace clic en un elemento, cuando una página se ha cargado o en el caso de que los campos de entrada se cambien.


Formularios

La validadión de un formulario se puede realizar con un script de Javascript, la cual puede ser llamada dentro del parametro onsubmit o en un boton dentro del elemento formulario en el HTML
